1. Variabile si constante

  • Variabila = "obiect"/"proprietate" care perimte stocarea unui tip de data in memoria calculatorului In cazul in care in program dorim sa folosim date avem nevoie de variabile Informal: variabila = sertar

  • Variabilele pot sa contina date ce se schimba sau date care raman constante (nu se schimba)

    • int x = 5 <- constanta
    • int x; cin>>x; <- changable

2. Tipuri de date

  • Indica ce fel de informatie se stocheaza intr-o variabila
  • Informal: Sertare de haine, prosoape, tricouri, etc. (haine, prosoape, tricouri = tiprui de date)
  • Tipurile de date pot sa fie de doua feluri:
    • simple:
      1. int/ unsigned int (numere intregi/intregi pozitive)
      2. long long (numere >10 cifre)
      3. char (charactere)
      4. float (numere cu zecimale)
      5. bool (1/0)
      6. double ("real", retine mai multe zecimale ca si float)
      7. pointer (! <-clasa a X-a)
    • derivate (! <- clasa a X-a)
  • Observatie: diferenta dintre tipurile de date, pe langa valorile efective pe care le stocheaza este si memoria pe care o ocupa (calculatorul are memorie limitata, de aceea incercam sa folosim doar memoria de care avem nevoie)
    • ex: nu vom folosi float x pentru a stoca un numar intreg, cu toate ca din punct de vedere matematic este corect.
  • Informal: nu vrem sa avem in camera sertare prea mari pe care niciodata nu le vom umple (ocupa spatiu)

3. Operatori

Operator de atribuire: =
  • "=" pune in variabila pe care o insoteste o anumita valoare:
    • ex: int a; a=3;

!!A nu se confunda cu ==!!

Operatori aritmetici: + - * / %

a:b=c,r ;a=b∗c+ra:b=c,r\space; a= b*c +ra:b=c,r ;a=b∗c+r

  • / indica catul impartirii (c)
  • % indica restul impartii (r) ! Foarte importante sunt /10 (elimina ultima cifra) si %10 (indica ultima cifra)
Operatori relationali: < > <= >= !=
Operatori logici: ! && ||
  • ! - NOT. Neaga valoarea de adevar a unei propozitii
  • || - OR. Returneaza o valoare adevarata daca una din propozitiile pe care le evalueaza este adevarata
  • && - AND. Returneaza valoarea de adevar daca ambele propozitii sunt adevarate
